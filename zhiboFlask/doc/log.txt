日志统计：
	数字：
		平台：
			1）平台运行成功（今天，昨天，前天）
			2）平台运行存在错误报告次数（今天，昨天，前天）

		爬虫：
			3）爬虫运行成功（今天，昨天，前天）
			4）爬虫数据存储错误次数（今天，昨天，前天）

	图表：
		平台：
			5）平台运行情况（总次数、成功运行、失败运行）（周，月）
			6）每次运行的时间（日、周、月）
			7）平台运行存在错误报告次数（日，周，月）
		爬虫：
			8）各爬虫的运行情况（总次数、成功运行、失败运行）（周，月）
			9）各爬虫的爬取处理情况（成功获取、保存成功、保存失败）（日，周，月）
			10）爬虫运行时间（总时间，爬取时间，保存数据时间）（日、周、月）

日志详情：
	文件目录树，直接打开查看


接口：

	log:
		platform:
			1）获取平台运行次数
			getRun(type,offset)
			param:
				type[string|{d,w,m}]:d-天;w-周;m-月;
				offset[int|{0-30}]:0表示今天（这周，这个月），1表示昨天（上周，上个月）
			return:
				type:list
				format:
					[
						{
							'date'[string|format:yyyy-MM-dd]:日期,
							'run'[int]:总次数,
							'success'[int]:成功次数,
							'failed'[int]:失败次数
						},
						{...}
					]
			2）获取平台运行存在错误报告次数
			getLogError(type,offset)
			param:
				type[string|{d,w,m}]:d-天;w-周;m-月;
				offset[int|{0-30}]:0表示今天（这周，这个月），1表示昨天（上周，上个月）
			return:
				type:list
				format:
					[
						{
							'date'[string|format:yyyy-MM-dd]:日期,
							'mount'[int]:次数
						},
						{...}
					]
			3）获取平台每次运行时间
			getRunTime(type,offset)
			param:
				type[string|{d,w,m}]:d-天;w-周;m-月;
				offset[int|{0-30}]:0表示今天（这周，这个月），1表示昨天（上周，上个月）
			return:
				type:list
				format:
					[
						{
							'date'[string|format:yyyy-MM-dd]:日期,
							'mount'[int]:次数,
							'time'[int]:多少秒
						},
						{...}
					]
		spider:
			4）获取爬虫种类
			getList()
			param:
			return:
				type:list
				format:
					[
						{
							'key'[string]:种类,
							'name'[string]:名称
						},
						{...}
					]
			5）获取各爬虫运行次数
			getRun(key,type,offset)
			param:
				key[string|{all,douyu,huajiao,panda,quanmin,yy,zhanqi}]:种类(all,返回所有)
				type[string|{d,w,m}]:d-天;w-周;m-月;
				offset[int|{0-30}]:0表示今天（这周，这个月），1表示昨天（上周，上个月）
			return:
				type:list
				format:
					[
						{
							'key'[string]:种类,
							'list'[list]:
								[
									{
										'date'[string|format:yyyy-MM-dd]:日期,
										'run'[int]:总次数,
										'success'[int]:成功次数,
										'failed'[int]:失败次数
									},
									{...}
								]
						},
						{...}
					]
			6）获取各爬虫的爬取处理情况
			getRunTime(key,type,offset)
			param:
				key[string|{all,douyu,huajiao,panda,quanmin,yy,zhanqi}]:种类(all,返回所有)
				type[string|{d,w,m}]:d-天;w-周;m-月;
				offset[int|{0-30}]:0表示今天（这周，这个月），1表示昨天（上周，上个月）
			return:
				type:list
				format:
					[
						{
							'key'[string]:种类,
							'list'[list]:
								[
									{
										'date'[string|format:yyyy-MM-dd]:日期,
										'mount'[int]:次数,
										'scrapy'[int]:爬取数量
										'success'[int]:成功保存数量
										'failed'[int]:失败保存数量
									},
									{...}
								]
						},
						{...}
					]
			7）获取各爬虫的爬取处理情况
			getRunTime(key,type,offset)
			param:
				key[string|{all,douyu,huajiao,panda,quanmin,yy,zhanqi}]:种类(all,返回所有)
				type[string|{d,w,m}]:d-天;w-周;m-月;
				offset[int|{0-30}]:0表示今天（这周，这个月），1表示昨天（上周，上个月）
			return:
				type:list
				format:
					[
						{
							'key'[string]:种类,
							'list'[list]:
								[
									{
										'date'[string|format:yyyy-MM-dd]:日期,
										'mount'[int]:次数,
										'time'[int]:多少秒
										'scrapyTime'[int]:多少秒
										'saveTime'[int]:多少秒
									},
									{...}
								]
						},
						{...}
					]
		logFile:
			8）获取文件目录树结构
			getTree()
			param:
			return:
				type:list
				format:
					[
						{
							'name'[string]:名称,
							'list'[list|opt]:下级结构,
							'open'[json|opt]:
								{
								'key'[string|{platform,douyu,huajiao,panda,quanmin,yy,zhanqi}]:查询种类
								'date'[string|format:yyyy-MM-dd]:日期
								}
						},
						{...}
					]
			9）获取文件内容
			getContent(key,date)
			param:
				key[string|{platform,douyu,huajiao,panda,quanmin,yy,zhanqi}]:查询种类
				date[string|format:yyyy-MM-dd]:日期
			return:
				type:json
				format:
					{
						'content'[string]:文本内容
					}
			10）获取log文件内容
			getLogContent()
			param:
			return:
				type:json
				format:
					{
						'content'[string]:文本内容
					}
